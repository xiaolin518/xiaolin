## 1. Product Overview
数据分析训练网站，为商务数据分析与应用专业学生提供Python代码学习环境和实战训练项目。
- 提供浏览器端Python代码执行环境，覆盖从基础数据处理到高级建模的10个实战项目
- 帮助学生掌握pandas、matplotlib、scikit-learn等主流数据分析工具，提升业务分析能力

## 2. Core Features

### 2.1 User Roles
无复杂用户角色区分，所有访客均可使用网站功能。

### 2.2 Feature Module
1. **首页**: 项目列表、导航菜单、网站介绍
2. **项目详情页**: 项目介绍、技术栈、预期产出、代码编辑器
3. **代码执行页面**: 在线Python编辑器、代码运行、结果展示

### 2.3 Page Details
| Page Name | Module Name | Feature description |
|-----------|-------------|---------------------|
| 首页 | 项目列表 | 10个数据分析项目卡片展示，包含项目名称、简介、技术栈 |
| 首页 | 导航菜单 | 顶部导航栏，包含网站logo、项目分类链接 |
| 项目详情页 | 项目介绍 | 详细描述项目核心任务、技术要求、学习目标 |
| 项目详情页 | 代码编辑器 | 内置Python代码编辑器，支持代码运行和结果展示 |

## 3. Core Process
用户访问网站 → 浏览项目列表 → 选择感兴趣的项目 → 在代码编辑器中学习和运行Python代码 → 完成项目学习

```mermaid
flowchart LR
  A[访问网站] --> B[浏览项目列表]
  B --> C[选择项目]
  C --> D[查看项目详情]
  D --> E[在编辑器中学习代码]
  E --> F[运行代码查看结果]
```

## 4. User Interface Design
### 4.1 Design Style
- 主色调：深蓝色 (#165DFF)，辅助色：青绿色 (#00B42A)
- 按钮风格：圆角矩形，带有轻微阴影效果
- 字体：Inter 作为主要字体，Fira Code 用于代码展示
- 布局风格：卡片式布局，清晰的模块划分
- 图标风格：使用 lucide-react 线性图标

### 4.2 Page Design Overview
| Page Name | Module Name | UI Elements |
|-----------|-------------|-------------|
| 首页 | 项目列表 | 响应式网格布局，卡片悬停效果，渐变背景 |
| 项目详情页 | 代码编辑器 | 深色主题代码区，浅色主题内容区，分栏布局 |

### 4.3 Responsiveness
采用响应式设计，桌面端优先，自适应平板和移动设备，触摸操作优化。
